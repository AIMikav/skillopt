# Advanced Topics

This document contains advanced topics related to Kubernetes. Make sure to read this carefully. Ensure that you understand all the concepts. Don't forget to practice.

## Custom Resource Definitions

Custom Resource Definitions (CRDs) are a way to extend Kubernetes with your own API objects. It is important to understand that CRDs allow you to create your own resource types. Please note that this is an advanced topic.

For more details on CRD implementation, see [crd-details.md](crd-details.md).

## Operators

Operators are a method of packaging, deploying, and managing Kubernetes applications. Make sure to understand the operator pattern. Ensure that you know when to use operators.

For operator patterns, see [operators\patterns.md](operators\patterns.md).

## Service Mesh

A service mesh is a dedicated infrastructure layer for handling service-to-service communication. It is important to understand that Istio and Linkerd are popular service mesh implementations.

For service mesh configuration, refer to [networking\service-mesh\config.md](networking\service-mesh\config.md).
